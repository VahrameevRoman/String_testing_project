1. Тест на обычные поля:
Вход: "Hello World"
Ожидаемый результат: {Hello, World}

2. Тест на поля в двойных кавычках и обычные поля:
Вход: "What" a wonderful "World"
Ожидаемый результат: {What, a wonderful, "World"}

3. Тест на поля в одиночных кавычках:
Вход: '12 'hfyvu78' &^(&%'
Ожидаемый результат: {"12 ", "hfyvu78", " &^(&%"}

4. Тест на экранирование кавычек и обратного слэша:
Вход: "Hello \"nice\' \\World"
Ожидаемый результат: {"Hello "nice' \World"}

5. Тест на строку со смешанными ковычками:
Вход: "I love 'programming' and coding"
Ожидаемый результат: {"I love 'programming' and coding"}

6. Тест на строку со смешанными ковычками:
Вход: "I love 'pro""gra'mmi"ng'' a"'nd coding"
Ожидаемый результат: {"I love 'pro", "gra'mmi", "ng", " a", "'nd coding"}

7. Тест на пустую строку:
Вход:
Ожидаемый результат: {}

8. Тест на строку с пустыми полями:
Вход:""
Ожидаемый результат: {}

9. Тест на строку с пробелом в ковычках и пробелами по краям строки вне ковычек:
Вход: " ' ' "
Ожидаемый результат: { }

10. Тест на строку с несколькими пробельными символами между полями:
Вход: Hi         there
Ожидаемый результат: {"Hi", "there"}
